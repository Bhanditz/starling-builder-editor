How to generate sprite sheet

Prerequisite:
Photoshop
ExtendScript Toolkit
TexturePacker or ShoeBox

1. Change export_psd.jsx and pack.py to set up your path
2. Open psd files
3. run export_psd.jsx from ExtendScript to generate the pngs from the psd
4. run ./pack.py -a ATLAS_NAME to generate the sprite sheets from the pngs
